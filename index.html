ODE REST API

Description of functions available from the Operational Data Environment (ODE) via RESTful interface. Ranges given follow interval notation, for example (0,1) or (0..1) means 0 < x < 1 and [0,1] or [0..1] means 0 ≤ x ≤ 1.

post /pdm
PDM Interface (pdmPost)
Submits a Probe Data Management message to ODE for distribution to the specified RSUs.

Request body

ProbeDataManagement ProbeDataManagement (required)
Body Parameter — PDM Message

Return type

array[Status]

Example data

Content-Type: application/json
[ {
  "success" : true,
  "message" : "aeiou"
} ]

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Probe Data Management message was submitted successfully. See the response body or ODE User Guide for details.

400

Bad Request. Issued if request is invalid or failed to be executed. See the response body or ODE User Guide for details.

500

Internal Server Error. See the response body or ODE User Guide for details.
